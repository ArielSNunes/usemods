## formatCurrency
Format numbers into local currency

```js [js]
formatCurrency(1234.56)
```

```html [template]
{{ formatCurrency(1234.56) }}
```

```html [returns]
$1,234.56
```

## formatTime
Format time into hours, minutes, and seconds

```js [js]
formatTime(3723)
```

```html [template]
{{ formatTime(3723) }}
```

```html [returns]
1hr 2min 3s
```
