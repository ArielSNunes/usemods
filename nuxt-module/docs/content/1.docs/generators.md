# Generators

A collection of generators

### generateShortId

::generateShortId
::

Generate a unique short ID based on the current timestamp

```js [js]
generateShortId(36)
```

**Returns:** 1HR2MIN3S

### generateInitials

::generateInitials
::

Generate initials from any string

```js [js]
generateInitials('John Doe')
```

**Returns:** JD
